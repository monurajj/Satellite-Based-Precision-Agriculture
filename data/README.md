# Data Directory

This directory stores data files generated by the pipeline or downloaded from external sources.

## Dataset statistics (default pipeline)

After running `python main.py` with **no existing `merged_data.csv`**, the pipeline generates **synthetic** Kansas-style data with these defaults (`main.py` → `generate_synthetic_dataset`):

| File | Rows (records) | Columns | Description |
|------|----------------|---------|-------------|
| **merged_data.csv** | **1,300** | **19** | One row per field × composite date (100 fields × 13 ten-day dates) |
| **features.csv** | **100** | **26** | One row per **field** after feature engineering (target + 25 numeric features) |

**Unit of prediction:** one row in `features.csv` = one field’s yield (tons/ha).

**Column groups (merged_data):** `field_id`, `date`, six Sentinel-2 bands (B2–B12 subset), four vegetation indices, `temperature`, `precipitation`, `solar_rad`, three soil columns, `yield`.

**Column groups (features):** `field_id`, phenology & weather (`ndvi_peak`, `season_length`, `cumulative_ndvi`, `ndvi_grain_fill_slope`, `precip_sum`, `temp_mean`, `solar_sum`, `gdd_sum`), soil (`soil_pH`, `soil_OC`, `soil_clay`), six bands, three PCA components, four interaction terms, `yield`.

*If you supply your own `merged_data.csv`, row counts will match your data; column counts follow the same schema.*

## Expected Files (Generated by Notebooks)

After running `01_data_acquisition.ipynb`, you should have:

- **raw_satellite_data.csv** – Sentinel-2 bands and vegetation indices per field/date
- **weather_data.csv** – ERA5 temperature, precipitation, solar radiation
- **soil_data.csv** – SoilGrids pH, organic carbon, clay content per field
- **merged_data.csv** – Combined dataset with all variables

## Pre-computed Data (Fallback)

If Google Earth Engine is not accessible, you may use pre-computed sample data:

1. **Synthetic data**: Run the synthetic data generation in `01_data_acquisition.ipynb` (cell that creates fallback data)
2. **Public bucket** (example): `gs://your-bucket/satellite-agriculture/sample_data.csv`

## Data Schema

| Column       | Type   | Description                          |
|-------------|--------|--------------------------------------|
| field_id    | str    | Unique field identifier              |
| date        | date   | Observation date                     |
| B2, B3, B4, B8, B11, B12 | float | Sentinel-2 band reflectance |
| NDVI, EVI, NDRE, SAVI | float | Vegetation indices       |
| temperature | float  | Mean daily temperature (°C)          |
| precipitation | float | Cumulative precipitation (mm)        |
| solar_rad   | float  | Solar radiation (MJ/m²)              |
| soil_pH, soil_OC, soil_clay | float | Soil properties   |
| yield       | float  | Target: tons per hectare             |
